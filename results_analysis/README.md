# Results

All evaluation outputs for the trained models. Selected plots are stored in the
repository; full plot sets and CSV summaries are auto-generated by the evaluation
scripts. The folder structure and key numbers are documented here.

|---|---|
| `roc_{E}GeV.png` | ROC curves — overall, τ vs jj, τ vs bb |
| `confusion_matrix_{E}GeV.png` | Confusion matrix at threshold 0.5 |

## BDT per-model plots

Each of the 6 BDT model folders contains:

| File | Description |
|---|---|
| `_bkg_rejection_curve.png` | 1/FPR vs signal efficiency (log-y), all 6 test energies |
| `_brej_vs_energy.png` | Background rejection at 4 working points vs √s |
| `_pt_efficiency_all_energies.png` | pT-binned τ tagging efficiency + fake rate, all 6 energies |
| `_tau_efficiency_vs_energy.png` | Jet-level and event-level τ efficiency vs √s |
| `_feature_importance.png` | Normalised gain feature importance |
| `_auc_vs_energy.png` | AUC (overall, τ vs jj, τ vs bb) vs test energy |
| `_perprocess_rejection.png` | Per-process rejection (τ vs jj, τ vs bb) all energies |
| `_metrics_table.txt` | Full metrics: AUC, Acc, Prec, Rec, F1, jet-eff, event-eff, BRej@30/50/70/90% |

**Metric definitions in the metrics table:**

- **Jet-eff**: fraction of tau jets (passing pT window) correctly tagged — equivalent to recall
- **Event-eff**: fraction of tau *events* where at least one jet is correctly tagged;
  this is the true event-selection efficiency relevant for H→ττ analyses
