# Results

All evaluation outputs for the trained models. Plots and CSV summaries are
auto-generated by the evaluation scripts and are **not stored in this
repository**. The folder structure and key numbers are documented here.

---

## Folder structure

```
results_analysis/
├── cnn/
│   ├── CNN_model1_125GeV/        ← training plots for CNN trained at 125 GeV
│   ├── CNN_model2_250GeV/        ← training plots for CNN trained at 250 GeV
│   ├── test_results_model1_125GeV/
│   │   ├── {energy}GeV/          ← per-energy plots (7 plots each)
│   │   ├── roc_all_energies_cnn.png
│   │   ├── metrics_vs_energy_cnn.png
│   │   ├── summary_cnn.csv
│   │   └── evaluation_log.txt
│   └── test_results_model2_250GeV/
│       └── (same structure)
├── vit/
│   └── (same structure as cnn/)
├── comparison/                   ← CNN vs ViT overlay plots
├── jet_images/                   ← average jet image visualisations
└── kinematics/                   ← kinematic distributions per energy
    └── {energy}GeV_{train|test}_data_results/
```

---

## Per-energy plots (generated for each model × each test energy)

| File | Description |
|---|---|
| `roc_{E}GeV.png` | ROC curves — overall, τ vs jj, τ vs bb |
| `confusion_matrix_{E}GeV.png` | Confusion matrix at threshold 0.5 |
| `score_dist_{E}GeV.png` | Classifier score distributions |
| `pt_dist_{E}GeV.png` | pT distributions of signal and background |
| `pt_efficiency_{E}GeV.png` | pT-binned tau efficiency and fake rates |
| `event_tagging_efficiency_{E}GeV.png` | Event-level tau tagging efficiency vs leading jet pT |
| `completeness_{E}GeV.png` | Fraction of tau jets tagged per event |
